# Arcs Arsenal

Tools and data for the board game [Arcs](https://ledergames.com/products/arcs).

## Features

- **Card Rankings** Win rates, picks, and stats for Leaders and Lore cards with interactive charts and analysis
- **Data Explorer** Detailed card statistics and insights
- **Dice Calculator** Probability calculator for dice rolls and combat outcomes
- **Custom Cards** Custom leaders & lore built with the [Arcs Leader Generator](https://github.com/Laurens1234/Arcs-Leader-Generator)
- **Tier List** Personal card rankings
- **Draft Helper** Tool to assist with drafting strategies

## Data Sources

- Stats from [Arcs League Tracker](https://docs.google.com/spreadsheets/d/13Wb-JoX7L2-o3Q-ejvepsx11MW--yhTN5oJ-I4Hp2DU/edit?gid=1136087345)
- Community data from BoardGameGeek
- Card images from [buriedgiantstudios/cards](https://github.com/buriedgiantstudios/cards)
